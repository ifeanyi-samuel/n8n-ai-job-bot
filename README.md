# n8n AI Job Bot 🤖

Tired of searching for remote jobs? 
This n8n workflow turns hours of job hunting into seconds.

## What it does
1. Scrapes RemoteOK for "AI Automation" jobs
2. Removes duplicates so you don't get spammed
3. Sends daily job alerts to Telegram

## Setup
1. Import AI-Job-Bot.json into n8n
2. Add your Telegram Bot Token + Chat ID
3. Set Cron to run 2x daily
4. Deploy and start getting jobs

## Tech Stack
n8n | RemoteOK API | Telegram Bot
